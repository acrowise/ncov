# 基于Vue-next + Vue-router-next 的webpack 工程项目

## 如何使用

    git clone https://github.com/veaba/ncov.git --checkout vue-next-template
    
    cd client
    
    npm install 
    
    npm run dev

## 版本

- vue@3.0.0-alpha-10
- vue-router@4.0.0-alpha-4
- webpack@4.x
